# Project Metrics Dashboard

A simple Java Spring Boot web application that displays software project metrics through a web dashboard.

## Overview

This application demonstrates the use of Spring Boot, MVC architecture, and Thymeleaf templates to render a dashboard page in the browser.

The dashboard can be extended to display project information such as:

- Total Services
- API Count
- Code Coverage
- Build Status
- Deployment Frequency
- Open Defects

## Technologies Used

- Java 17
- Spring Boot
- Maven
- Thymeleaf
- Git
- GitHub

## Project Structure

src/main/java
- ProjectMetricsDashboardApplication.java
- ProjectMetricsController.java

src/main/resources/templates
- index.html

src/main/resources
- application.properties

## Features

- Spring Boot application setup
- MVC controller implementation
- Thymeleaf template rendering
- Web page served at `http://localhost:8080`
- GitHub integration for source control
